# Portfolio Screenshots

This directory should contain screenshots of portfolio projects.

## Required Files

- `vodacihronov.png` - Screenshot of https://vodacihronov.com/
- `sea-kayaking.png` - Screenshot of https://sea-kayaking.netlify.app/

## Image Guidelines

- Recommended size: 1200x800 pixels
- Format: PNG or JPG
- File size: Keep under 500KB for optimal loading

The application will automatically fall back to a placeholder image if screenshots are not available.
